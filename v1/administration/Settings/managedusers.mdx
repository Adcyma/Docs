The Managed Users page shows all users currently managed by Adcyma. From here you can monitor user information, control which users are in scope, and keep user data synchronized with your identity provider.

## User Information Display

The table shows one row per managed user with sortable columns:

- Display Name
- Email
- Department
- Job Title

Use the search bar to filter users by name, email, department, or job title. A badge at the top shows the total count of managed users.

## Managing User Scope

Select the Entra ID group that defines which users Adcyma manages from the dropdown menu. This group determines the full scope of managed users. After selecting your group, click **Save Group** to apply.

:::warning Important Group Type Requirement
The selected Entra ID group **must be a standard security group** and **cannot be a dynamic group**. Dynamic groups prevent Adcyma from updating group membership correctly. Always use manually managed or rule-based static groups.
:::

## User Data Synchronization

Adcyma automatically syncs user information from your identity provider every hour, so user details, group memberships, and organizational changes stay up to date.

If you need an immediate update, click **Refresh Users** to trigger a manual sync. This is handy when you've just added new users to the managed group, updated user info in your identity provider, or need to verify recent changes right away.

## Best Practices

- Choose standard Entra ID security groups that reflect the users you want Adcyma to manage.
- Avoid dynamic groups. Use only static security groups so Adcyma can update group membership properly.
- Review group membership regularly to make sure it matches your actual needs.
- Use manual refresh when you need immediate updates. For routine changes, the automatic hourly sync handles things on its own.
