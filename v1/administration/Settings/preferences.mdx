The Preferences page lets you set formatting rules for how Adcyma creates and displays user identity information. These settings keep things consistent across all new users and align with your naming conventions and email policies.

## Configurable Format Options

You can customize the format for these user attributes:

**User Principal Name (UPN)**
How the primary login identifier is structured for new users.

**Email Address Format**
How email addresses are constructed for new users.

**Display Name Format**
How user names appear in the system and in communications.

**Email Nickname Format**
The format for email aliases and short identifiers.

## Configuration Process

Use the dropdown menus to select your preferred format for each attribute. After making your selections, click **Save Preferences** to apply the changes.

:::info Important Note
These formatting preferences **only apply to newly created users** and will not modify the format of existing user accounts. Current users will retain their existing UPN, email, display name, and nickname formats until manually updated.
:::

## Tips

Set these preferences early in your Adcyma deployment so all new users follow consistent naming patterns from the start. Make sure the formats you choose work with your existing email domains and directory structures, and that they'll still make sense as your organization grows.
